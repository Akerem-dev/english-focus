# Final Product Specification

The final V1 is a local personal English vocabulary library.

## Primary screens

- Vocabulary
- Library
- Settings

## Missing-word flow

Search -> not found -> copy external-AI instruction -> paste JSON -> parse -> validate -> review issues -> preview -> resolve duplicate -> add to library.

## Local data layers

- Core vocabulary
- User vocabulary
- User overrides
- User metadata

## User metadata

- Favorite
- Tags
- User notes
- Learning status
- Review status
- Last viewed
- View count
- Created and updated timestamps

## Explicit exclusions

No API provider, API key, embedded AI, Ollama, local model, PDF RAG, cloud sync, account, subscription, gamification, or future learning screens.
