# CP16 — Single-entry file transfer

CP16 connects local `.json` files to the existing validated import workflow and adds direct export from Vocabulary detail.

## Included

- active top-bar Import action on all three routes;
- local `.json` selection without upload;
- extension, size, syntax, and detected-word gate;
- prefilled schema, semantic, preview, duplicate, and persistence workflow;
- direct single-entry JSON export from Vocabulary detail;
- saved-entry deep link after file import.

## Excluded

- multi-entry vocabulary packs;
- streaming pack progress;
- import history;
- backup and restore.
