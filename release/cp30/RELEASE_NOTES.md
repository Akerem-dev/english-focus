# English Focus 1.0.0

English Focus is an offline-first Windows vocabulary workspace for creating, validating, organizing and studying rich English vocabulary entries.

## Included in V1

- Rich vocabulary detail pages with meanings, grammar, examples, collocations, word families, related words, common mistakes and etymology.
- Safe single-entry and vocabulary-pack JSON import flows with schema validation, semantic checks, preview and duplicate handling.
- SQLite persistence for user vocabulary, core overrides, personal study metadata and application settings.
- Searchable and filterable Library with selection, copy and export tools.
- Local backup, validation, restore, retention and pre-restore safety backups.
- Local diagnostics, database health checks and non-destructive maintenance guidance.
- Command bar, keyboard shortcuts, accessible modal focus handling and route navigation.
- Toast feedback, favorite undo and standardized user-facing errors.
- Privacy-safe local activity history and selective local-data reset controls.
- Bundled verified core vocabulary content with content-quality gates.
- Windows NSIS and MSI installers with upgrade and data-preservation rehearsal.

## Privacy

English Focus is local-first. Vocabulary data, personal notes, study metadata, settings, activity records and backups remain on the device. No telemetry or paid external API is included.

## Upgrade guarantee tested for V1

The 0.9.0 release candidate was tested against legacy SQLite data, old backup formats, NSIS/MSI upgrade, reinstall, uninstall data preservation and downgrade blocking before promotion to 1.0.0.

## Signing notice

The provided installer is unsigned unless a separate code-signing certificate was applied during packaging. Windows may display an Unknown Publisher or SmartScreen warning for unsigned builds.
